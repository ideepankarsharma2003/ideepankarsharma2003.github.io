Chicken Disease Classification using Fecal Images

"Chicken Disease Classification using Fecal Images" is a remarkable project showcased in my portfolio, employing an innovative approach to detect and classify diseases in chickens through fecal images. The project utilizes a Flask web application to offer user-friendly interactions. Leveraging Deep Version Control (DVC) pipelining, the application encapsulates different stages, ensuring seamless data processing. The DVC pipeline comprises stages such as data ingestion, preparing the base model, training, and evaluation. The intelligent pipeline systematically processes the fecal images through each stage, facilitating data augmentation, model training, and evaluation. The application showcases the power of integrating modern web technologies with advanced machine learning techniques, ultimately contributing to more efficient disease detection and management in poultry farming."
